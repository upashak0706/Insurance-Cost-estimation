Medical Insurance Cost Predictor | Python, Scikit-Learn, Pandas

Developed a predictive model using Linear Regression to estimate individual medical insurance charges with an evaluation focused on R-squared and Adjusted R-squared metrics.

Performed Exploratory Data Analysis (EDA) using Seaborn and Matplotlib to identify key cost drivers, such as smoking status and BMI.

Implemented data preprocessing pipelines including StandardScaler for feature normalization and categorical encoding for demographic data.
